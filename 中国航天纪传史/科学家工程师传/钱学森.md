# 钱学森

## 基本信息
+ **生卒年**：  
1911年12月11日－2009年10月31日
+ **出生地**：  
上海
+ **祖籍**：  
浙江省杭州市
+ **身份**：  
中国著名的科学家和工程师
+ **基本经历**：  
1934年毕业于上海交通大学，后赴美国加利福尼亚理工学院攻读航空工程，1938年获得博士学位,1955年回国，长期担任火箭导弹和航天器研制的技术领导职务  
是中国航天事业的奠基人之一，被誉为“中国航天之父”和“中国导弹之父”，在空气动力学、喷气推进和系统工程等领域做出了卓越贡献
